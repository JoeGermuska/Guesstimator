About

Guesstimator was made for #ONA14newsgames by Sisi Wei and Rebekah Monson using the supercool Census Reporter API.

It's built using basic HTML, CSS and javascript so that it is easy to implement for your city on your site. To make your own Guesstimator, fork this project.
